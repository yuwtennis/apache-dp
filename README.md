
# Overview

This repository sample configurations for kafka related products.

# How to start kafka

## Local

1. Start Broker

https://kafka.apache.org/quickstart

2. Start Kafka Connect as standalone mode

https://kafka.apache.org/documentation/#connect_running

## Docker
```
docker-compose up -d
```
